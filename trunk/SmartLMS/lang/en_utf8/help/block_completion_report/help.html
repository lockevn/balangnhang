Completion Report

This block allows an Administrator, a Teacher, or an Editing Teacher to define course completion requirements in a rubric of four sections, i.e. assignments, quizzes, resources, and manual grade. The Administrator has access to all the courses, while the Teachers can only access the courses that they teach.

The assignment grade can be selected from a chosen final project or the average of all assignment. The quiz grade can be similarly selected from a final exam or the average of all quizzes. The resources grade is simple complete or incomplete depending on the list of required resources that the student must visit (such as opening a Word document, PDF file, or visited a web site). The last section is a manually set grade.

The four sections are displayed as columns in a report for each course. If a section is not required for a course, the column is not displayed.

There are six php files in this block:

  1. block_completion_report - standard block file
  2. configure.php - configure Completion Requirement Rubric
  3. report.php - display completion status for all participants in a selected course
  4. participant.php - display completion status for a selected participant in a selected course
  5. accessed_resources.php - displays the resources in a course which have been accessed by a participant
  6. required_resources.php - displays the resources which are required to complete a course

configure.php

Use this page to set the rubric of criteria that defines Completion Requirement of a course
Linked from:
block_completion_report.php in the block - 'Configure Requirement Rubrics'
report.php at the top of the page - 'Configure Requirement Rubrics for this course'

Each course has its own 13 configuration settings stored in the database table completion_configure The 13 settings can be divided in four areas, in addition to the course id:

  1. intCourseID - course id in the table 'course'

    2. Assignment

  2. enable - on, off
  3. percentage grade - 0 to 100
  4. grade method - AverageAllAssignments or FinalProject
  5. FinalProject - assignment id from table 'assignment'

    6. Quiz

  6. enable - on, off
  7. percentage grade - 0 to 100
  8. grade method - AvaregeAllQuizzes or FinalExam
  9. FinalExam - quiz id from table 'quiz'

    10. Resources

  10. enable - on, off
  11. Current Required Resources - string of resource id (from table 'resource'), separated by comma

    12. Manual

  12. enable - on, off
  13. percentage grade - 0 to 100

report.php

Use this page to display the completion status of all the participants in a selected course.
Linked from
block_completion_report.php in the block - Generate Report
configure.php at the top of the page - 'Display Completion Report for this course'

Each participant has a record in the the database table completion_report:

  • userid - id from table 'user'
  • userfirstname - firstname from table 'user'
  • userlastname - lastname from table 'user'
  • intaverageQuiz - percentage grade - 0 to 100
  • intGradeFinalExam - id from table 'quiz'
  • intaverageAssignment - percentage grade - 0 to 100
  • intGradeFinalProject - id from table 'assignment'
  • intGradeManual - percentage grade - 0 to 100
  • strCompletedResources - 'Completed' or 'Incomplete'
  • courseid - id from table 'course'
  • strCompletionStatus - 'Completed' or 'Incomplete'

participant.php

Use this page to display the completion status of a selected participant in a selected course. Also use it to assign a manual grade to the participant.
Linked from
report.php in the report table - first and last name of the participant
