Path to Gallery2 Directory

This is the location to your Gallery2 directory. The path can be an absolute path or it can be relative to the root directory of your Moodle installation

For example:

Your moodle install is located here:

  • /home/example/public_html

And you have installed Gallery2 in another directory:

  • /home/example/public_html/gallery2

So the two paths that you could enter are:

  • the relative path: /gallery2/
  • or the absolute path: /home/example/public_html/gallery2/

Note: the trailing / at the end of the path is required.
