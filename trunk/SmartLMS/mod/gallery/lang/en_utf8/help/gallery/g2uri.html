Path

URI of the G2 standalone location. Path from docroot to the Gallery2 directory where main.php is located. Protocol/hostname are both optional. Please include a trailing slash (/).

Example:

Gallery2 location: http://www.mysite.com/subdir/gallery2/

URI can be either of the two following:

  • /subdir/gallery2/
  • http://www.mysite.com/subdir/gallery2/
