Linked Album

This is the root Gallery album that this Moodle Gallery Module is linked to. If an album has already been linked to another Moodle Gallery Module, then that album does not appear in the list.

When changing from one album to another, then the behavior of the Album title, Album summary, Album description, and Album keywords changes slighly.

  • If any of the above four fields are not empty, then the newly linked album will be updated with the field value.
  • If any of the above four fields are empty, then the newly linked album retains its original value for that field.

Examples:

If the Album summary is left blank, then the newly linked album's summary will retain its original value.

If the Album summary has text, then the newly linked album's summary will be changed to the entered text.

When not changing from one album to another, then whatever value entered for those four fields will be set in the linked album.

Example:

If the Album summary is left blank, then the linked album's summary will be set to nothing.
