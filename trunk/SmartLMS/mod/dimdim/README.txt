Dimdim Web Meeting module for Moodle:
------------------------------------

Dimdim (www.dimdim.com) is Open Source web meeting product. With Dimdim you can show Presentations, Applications and Desktop to 
any other person over the Internet. You can chat, show your webcam and talk with others in the meeting. 

The Dimdim Web Meeting activity for moodle is still buggy and being worked on, but if you want to test it and help out, here are some quick instructions:

1.Deploy the folders provided in zip file to the corresponding folders in moodle installation
2.Login as admin and click on Notifications to create the database tables for Dimdim Web Meeting activity
3.Go to settings of Dimdim Web Meeting and provide the Dimdim Web Meeting server’s hostname and the port number. If you do not have a Dimdim Web Meeting server installed locally, you can provide www1.dimdim.com as host name and 80 as port number.
4.Configure your time zone to your local time zone to ensure that the Dimdim Web Meeting module works properly.

For detailed information, please download the installation guide and user guide from http://sourceforge.net/projects/dimdim/
For more information, visit http://www.dimdim.com/ or mail us at rajesh@dimdim.com
